# OurApp 

OurApp is a simple productivity and motivation mobile application built using React Native (Expo).

The app displays the current time, allows the user to enter their name, and provides a daily motivational quote to encourage consistency and positive habits it also change background ?

---

##  Features

-  Live Time Display
-  User Name Input
-  Personalized Welcome Message
-  Daily Motivational Quote ai generated
-  Cross-platform support (Android & iOS via Expo)
-  ability to change to dark mode
---

##  Tech Stack

- React Native (Expo)
- JavaScript
- React Hooks (useState, useEffect)
- GitHub

---

##  Installation & Setup

### 1 Clone the repository

```bash
git clone https://github.com/alinassir23424/ali_nassir_task
cd OurApp
```

### 2 Install dependencies

```bash
npm install
```

### 3 Start the development server

```bash
npx expo start --tunnel
```

Scan the QR code using Expo Go on your mobile device.
